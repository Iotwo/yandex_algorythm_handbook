Полное решение

Бронирование переговорки

Ограничение времени

1 с

Ограничение памяти

256.0 Мб

Ввод

стандартный ввод или input.txt

Вывод

стандартный вывод или output.txt

Задано n n n интервалов. Требуется найти максимальное количество взаимно непересекающихся интервалов.

Два интервала пересекаются, если они имеют хотя бы одну общую точку.

Формат ввода

В первой строке задано одно число n n n ( 1 ≤ n ≤ 100 ) (1 \leq n \leq 100) (1≤n≤100) — количество интервалов.

В следующих n n n строках заданы интервалы l i , r i l_i, r_i li​,ri​ ( 1 ≤ l i ≤ r i ≤ 50 ) (1 \leq l_i \leq r_i \leq 50) (1≤li​≤ri​≤50).

Формат вывода

Выведите ответ на задачу.

Пример 1

Ввод
3
1 3
2 3
4 5
Вывод
2

1 2
2 3
